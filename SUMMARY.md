# Project Summary: AI CV Reviewer

## Overview
Developed a Python-based AI CV Reviewer application using Streamlit and Google's Gemini API.

## Features
- **PDF Text Extraction**: Uses pdfplumber to extract content from CVs.
- **Gemini 1.5 Integration**: Analyzes CVs against job descriptions.
- **Analysis Metrics**: Provides a Match Score, keyword analysis, formatting checks, and STAR method evaluation.
- **Interactive UI**: Clean Streamlit interface with sidebar configuration.

## Technical Details
- **Location**: D:\App\CVreviewerAPP
- **Primary Language**: Python
- **Main Branch**: main
- **Security**: .env file for API keys, excluded via .gitignore.

## Next Steps
1. Run 'streamlit run app.py' to start the application.
2. Enter Gemini API key in the sidebar or .env file.
3. Upload CV and paste Job Description for analysis.
